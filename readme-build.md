# Expo SDK Update

## To update Expo SDK

> npm install expo@latest

> npx expo install --fix

## Fix typescrip issues:

> Ctrl+Shift+P > Typescript: Restart TS Server

## Expo Doctor

> npx -y expo-doctor

## Build

> eas build --platform android

## Run

> npm run web
